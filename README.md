# Used Car Pricing API

- Users sign up with email / password
- Users get an estimate for how much their car is worth based on the make/model/yesy/mileage
- Users can report what they sold their vehicles for
- Admins have to approve reported sales


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
